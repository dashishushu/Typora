## 侧边栏区

### 显示/隐藏 `左侧边栏`

> Ctrl + Shift + L

### 显示/隐藏 `当前文件大纲`

> Ctrl + Shift + 1

### 显示/隐藏 `文档列表`

> Ctrl + Shift +2

### 显示/隐藏 `文件树`

> Ctrl + Shift + 3

## 显示模式

### 进入/退出`源代码模式`

> Ctrl + /

### 进入/退出`专注模式`

> F8

### 进入/退出`打字机模式`

> F9

### 进入/退出`全屏模式`

> F11

### 放大文字

> Ctrl + Shift + =

### 缩小文字

> Ctrl + Shift + -
